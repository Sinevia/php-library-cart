## Installation ##

Add the following to your composer file:

```json
   "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/sinevia/php-library-cart.git"
        }
    ],
    "require": {
        "sinevia/php-library-cart": "dev-master"
    },
```
